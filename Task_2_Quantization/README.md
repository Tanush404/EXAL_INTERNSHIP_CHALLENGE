## Task 2: Quantization & Accuracy Analysis
### Dataset
The evaluation was performed using the **Tiny ImageNet** dataset (200 classes), which can be downloaded here:
* [Tiny ImageNet Dataset](http://cs231n.stanford.edu/tiny-imagenet-200.zip)

### Requirements
* **Flat Structure:** All validation images must be in `val/images`.
* **Tab Separation:** Labels must be parsed from `val_annotations.txt` using `split('\t')`.
